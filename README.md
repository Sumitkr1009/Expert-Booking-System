# Expert Booking System

A full-stack Expert Booking System built using React, Node.js, Express, and MongoDB.

## Features

-  Search & Filter Experts
-  Pagination (5 experts per page)
-  Expert Detail Page
-  Slot Booking System
-  Prevent Double Booking
-  Real-time Slot Update using Socket.io
-  My Bookings (Search by Email)
-  Frontend & Backend Validation
-  Clean and Responsive UI

---

## Tech Stack

### Frontend
- React (Vite)
- React Router
- Axios
- CSS
- Socket.io Client

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.io


## Installation

### Backend
cd backend
npm install
npm run dev

Create `.env` file:
PORT=5000
MONGO_URI=mongodb://localhost:27017/expertBookingDB


### Frontend
cd frontend
npm install
npm run dev
